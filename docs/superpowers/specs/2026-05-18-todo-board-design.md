# To Do List 看板设计说明

## 目标

构建一个铺满整个浏览器视口的 To Do List 看板。页面不产生整体滚动，所有文本、图片和按钮配置都保存在浏览器 `localStorage` 中。

## 文件结构

- `index.html`：页面结构。
- `styles.css`：布局、主题、弹窗、预览层和响应式细节。
- `app.js`：状态管理、保存逻辑、图床、快捷按钮、待办事项和剪贴板交互。
- `README.md`：中文使用说明。

## 页面布局

页面使用五栏横向布局，宽度比例为 `10% / 20% / 20% / 25% / 25%`。

1. 左侧 10%：图床区域。
2. 右侧 20%：便签与快捷按钮区域，二者各占垂直高度 50%。
3. 再右侧 20%：待办事项区域，垂直分为早、中、晚。
4. 再右侧 25%：工作空间。
5. 最右侧 25%：工程文件/工作存储区。

页面整体采用纯白背景、简单线条、低装饰风格，并支持白天/黑夜模式切换。主要区域采用紧凑线框布局，面板、标题、textarea 和列表之间不保留额外空白，只用连续单线分隔。

## 图床区域

- 支持通过 `Ctrl+C`、`Ctrl+V` 粘贴图片。
- 图片以缩略图形式从上到下排列。
- 图片以 Data URL 保存到 `localStorage`。
- 单击图片进入预览。
- 双击图片复制图片到剪贴板。
- 预览时支持鼠标滚轮或触控板缩放。
- 预览时支持鼠标按住拖动图片。
- 预览快捷键：
  - `Delete`：删除当前图片。
  - `Enter`：复制当前图片。
  - `Space`：关闭预览。

## 便签与快捷按钮区域

便签为一个 textarea，支持延迟自动保存和 `Ctrl+S` 手动保存。

快捷按钮支持两种类型：

1. 链接属性：
   - 字段为标题和 URL。
   - 页面只显示标题按钮。
   - 点击按钮复制 URL，不跳转。

2. 复制文本属性：
   - 字段为标题和复制文本。
   - 页面只显示标题按钮。
   - 点击按钮复制文本。

按钮交互：

- 支持新增按钮。
- 新增按钮通过模态框完成，不在快捷按钮区常驻新增表单。
- 右键按钮打开菜单。
- 菜单支持编辑、隐藏/取消隐藏、删除。
- 隐藏后默认不显示。
- 快捷按钮区域提供“显示全部/隐藏已隐藏”切换，可显示包含隐藏按钮在内的全部按钮。
- 所有删除操作都会先弹出二次确认，用户取消时不删除数据。

## 待办事项区域

待办事项分为早、中、晚三块：

- 每块支持点击空白处新增待办事项。
- 空白新增的待办事项如果没有输入文字，离开编辑状态时自动丢弃，不写入 `localStorage`。
- 单击任意待办事项可直接编辑。
- 编辑待办事项时按回车，会在当前项下方新增一条待办事项。
- 支持勾选完成。
- 标记完成的待办事项自动移动到所在块底部。
- 支持右键打开菜单，再从菜单中选择删除。
- 支持在同一块内拖拽排序。
- 支持从一个块拖动到另一个块。
- 支持删除，所有删除操作都会先弹出二次确认。
- 每块右上角支持一键清除已完成事项，清除前弹出二次确认。
- 数据保存到 `localStorage`。

## 工作空间与工程文件/工作存储区

两栏均为整栏 textarea：

- 支持长文本输入。
- 支持延迟自动保存。
- 支持 `Ctrl+S` 手动保存。

## 保存逻辑

适用于所有 textarea：

- `Ctrl+S`：立即保存全部数据。
- `Ctrl+S` 只显示视频头像和保存气泡，不额外显示 toast 文案。
- 用户输入文字后，若 3 秒内没有继续输入，则自动保存。
- 工作空间输入后 3 秒延时保存；若 3 秒内工作空间失焦，则立即保存并取消延时保存定时任务。
- textarea 失焦后，1 秒后自动保存。
- 如果 textarea 失焦后 1 秒内另一个 textarea 获得焦点，则取消这次失焦保存。
- textarea 聚焦时，在 textarea 右下角显示 `static/video/hermes.mp4` 方形动态头像。
- 没有 textarea 聚焦时，`Ctrl+S` 保存会在页面右下角显示动态头像和保存气泡。
- 动态头像隐藏时保持播放，不因 textarea 失焦而暂停。
- 保存触发时，在视频正上方显示气泡。
- 气泡内容由 10 条随机保存文案和 10 个随机颜文字组合。
- 气泡显示 3 秒后消失。

## 存储策略

所有状态集中保存在 `localStorage` 的 `todo-board-state-v1` 键中。保存内容包括：

- 主题模式。
- 图床图片。
- 便签文本。
- 快捷按钮配置。
- 快捷按钮显示隐藏开关。
- 早、中、晚待办事项。
- 工作空间文本。
- 工程文件/工作存储区文本。
