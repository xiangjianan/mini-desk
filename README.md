# To Do List 看板

这是一个静态 To Do List 看板，使用 `index.html`、`styles.css` 和 `app.js` 实现。页面铺满整个浏览器视口，不依赖前端框架，数据保存在浏览器 `localStorage` 中。

## 使用方式

直接打开 `index.html` 即可使用。也可以在目录中启动一个本地静态服务器后访问页面。

## 功能说明

- 图床区支持粘贴图片、预览、缩放、拖动、删除和复制图片。
- 便签和快捷复制区域各占所在栏 50% 高度。
- 便签区、工作空间、工程文件区均支持文本输入和自动保存。
- 快捷按钮支持复制 URL 或复制文本，按钮只显示标题。
- 快捷按钮通过图标新增按钮打开模态框创建，不在页面中常驻新增表单。
- 新增/编辑快捷按钮时，通过勾选框设置是否为链接属性。
- 快捷按钮采用带间距的白色线框按钮。
- 快捷按钮支持右键编辑、隐藏、取消隐藏和删除。
- 待办事项区分为早、中、晚三块，支持点击空白处新增、单击编辑、回车续写、完成、右键菜单删除、拖拽排序和跨区域拖动。
- 每个待办模块右上角支持一键清除已完成事项。
- 空白新增的待办如果没有输入文字，离开编辑状态时会自动丢弃，不会保存。
- 标记完成的待办事项会自动移动到所在区域底部。
- 所有删除操作都会先弹出二次确认。
- 支持白天模式和黑夜模式切换。
- 页面为紧凑线框布局，主要区域之间只使用单条分隔线。

## 图床快捷键

点击图片进入预览后：

- `Delete`：删除当前图片。
- `Enter`：复制当前图片。
- `Space`：关闭预览。
- 鼠标滚轮或触控板：放大、缩小图片。
- 鼠标按住拖动：移动图片。

## 保存逻辑

- `Ctrl+S`：立即保存全部数据，只显示视频头像和保存气泡。
- textarea 输入停止 3 秒后自动保存。
- 工作空间输入后 3 秒延时保存；如果 3 秒内失焦，则立即保存并取消这次延时保存。
- textarea 失焦 1 秒后自动保存。
- 如果失焦后 1 秒内聚焦到另一个 textarea，则取消这次失焦保存。
- textarea 聚焦时，右下角显示 `static/video/hermes.mp4` 动态头像。
- 没有 textarea 聚焦时，`Ctrl+S` 保存会在页面右下角显示动态头像和保存气泡。
- 动态头像隐藏时不会暂停播放。
- textarea 失焦后，动态头像位置恢复到屏幕右下角。
- 动态头像和保存气泡出现、消失时带渐变效果。
- 保存时显示“已保存”语义的随机气泡消息，3 秒后消失。

## 数据位置

数据保存在浏览器 `localStorage` 中，键名为：

```text
todo-board-state-v1
```

清空浏览器站点数据或手动删除该键会清除看板内容。
