# Mini Desk

Mini Desk 是一个本地优先的个人桌面工作台，用来集中整理截图、备忘录、提醒事项、快捷动作、工作空间和常用工具。它基于 Vue 3、TypeScript、Vite、Naive UI 和轻量 shadcn 风格组件构建，界面风格参考 Apple Human Interface Guidelines，强调紧凑、清晰、低干扰的日常使用体验。

页面状态保存在浏览器 `localStorage`，图片原文和自定义 GIF 资源保存在同站点 IndexedDB。默认不依赖后端服务，适合作为个人浏览器工作台使用。

## 在线地址

- <https://minidesk.online>

## 快速开始

安装依赖：

```bash
npm install
```

启动本地开发服务：

```bash
npm run dev
```

如果需要在局域网内用其他设备访问：

```bash
npm run dev -- --host 0.0.0.0
```

构建生产产物：

```bash
npm run build
```

运行测试：

```bash
npm test
```

## 功能概览

### 工作台体验

- 四栏桌面工作台布局，集中展示图床、工具、快捷动作、提醒事项和工作空间。
- 支持拖动区域分隔线调整各区域宽度，宽度配置会自动保存。
- 顶部 Header 支持收起和展开，减少占用空间。
- 浅色/深色主题、半透明材质、统一图标尺寸和紧凑控件风格参考 Apple HIG。
- 主要浮层、菜单、预览和编辑态统一使用低对比边框与柔和背景，避免混用多套组件视觉语言。

### 图床

- 支持 `Ctrl+V` 粘贴截图。
- 支持把浏览器外部图片拖入图床。
- 图片原文写入 IndexedDB，页面状态只保存图片元数据。
- 图片列表支持缩略图预览、拖拽排序、右键菜单、删除和复制。
- 图片预览默认按原始像素尺寸展示；当图片超过页面可视区域时，会按页面最大宽高自适应约束。
- 预览模式支持点击空白处关闭、侧边栏切换图片、右键菜单、滚轮缩放和拖动平移。
- 预览快捷键：`Enter` 复制、`Delete` 或 `Backspace` 删除、`Space` 或 `Esc` 关闭、方向键切换图片。

### 便签

- 使用原生 textarea，点进去即可输入。
- 支持换行、Tab 缩进、Shift+Tab 取消缩进。
- 删除行文字后继续按删除键，可以取消该行缩进。
- 文本区为空时，聚焦会显示 Tips 气泡和动态头像；非空时只显示动态头像。

### 快捷链接

- 支持链接型按钮和复制文本型按钮。
- 支持快捷 API 请求按钮，可配置请求方法、URL、Header、Body 和响应复制策略。
- 复制文本型按钮支持多行内容。
- 按钮点击后打开链接或复制文本。
- 右键菜单支持编辑、隐藏、显示和删除。
- 顶部三点菜单支持新增按钮、显示隐藏项、收起隐藏项和 Tips。
- 支持拖拽排序。

### 工具

- 工具区内置计算器、进制转换、取色板、编解码和随机密码生成。
- 计算器支持 `+`、`-`、`×`、`÷`、`%`、小数、退格和键盘输入。
- 进制转换支持二进制、八进制、十进制和十六进制互转，并支持一键复制结果。
- 取色板支持手动选色、浏览器取色笔，以及 HEX、RGB、HSL 三种格式复制。
- 编解码支持 Base64 和 URL 编码/解码。
- 随机密码生成支持长度、字符类型和自定义特殊字符集。
- 工具可以在配置面板中显示或隐藏，当前打开的工具会自动记忆。

### 提醒事项

- 按早上、中午、晚上三个区域组织提醒事项。
- 单击空白处新增提醒事项，单击条目即可编辑。
- 支持回车续写、完成、删除、复制文本、拖拽排序和跨区域移动。
- 每个时间段标题旁显示完成进度，例如 `2/5`。
- 已完成提醒事项默认隐藏，可按区域单独显示或隐藏。
- 完成项前会显示弱分隔标题“已完成”。
- 点亮星标时会在星标旁打开截止日期和常用整点选择，默认选中当前时间之后的下一个常用整点；可选择截止时间后确认，也可忽略时间只设为重点。
- 有截止时间的星标事项会在所属时间段内按截止时间靠前排序。
- 截止时间会显示在星标前，临近和超期事项会用中文时段标签、背景底色和左侧警示线提醒。
- 点击截止时间选择框外部会关闭选择框，不会自动设为重点。
- 取消星标前会二次确认；确认后会同步清除该事项的截止时间。
- 支持清理已完成事项，清理前会弹出二次确认气泡。

### 今日重点

- 星标提醒事项会在所属区域置顶，并同步出现在“重点事项”区域。
- 今日重点会显示未完成和已完成的星标事项，已完成项弱化展示。
- 今日重点标题支持双击改名，默认标题为 `重点事项`。
- 今日重点中的未完成条目按截止时间靠前排序，无截止时间的重点事项排在后面，已完成项继续弱化展示。
- 今日重点中的条目支持右键设置或编辑提醒时间，但不参与拖拽排序。

### 工作空间

- 右侧区域合并为工作空间 Tab。
- 初始只有一个“工作空间”，系统至少保留一个空间。
- 支持新增空间、双击重命名、右键删除、拖拽排序。
- 删除空间前会二次确认；删除后不可恢复。
- 空间过多时，Tab 栏支持横向滚动。
- 工作空间文本区点进去即可输入，支持与便签一致的缩进体验。

### 设置与反馈

- 顶部显示保存状态：已保存、保存中、有未保存内容。
- 设置菜单支持数据导入、数据导出、语言切换、GIF 主题、快捷键帮助、提建议、关于和版本状态。
- GIF 主题支持内置主题、隐藏和自定义浅色/深色 GIF。
- “提建议”会打开 GitHub issue 创建页面。
- “关于”通过消息气泡显示项目简介，并提供可点击的 GitHub 项目链接。
- 支持浅色和深色主题切换，主题选择会自动保存。

### 快捷键

- `Ctrl+S`：立即保存当前看板。
- `Ctrl+V`：在图床区域粘贴图片。
- 图片预览：`Enter` 复制，`Delete` / `Backspace` 删除，`Space` / `Esc` 关闭，方向键切换图片。
- 文本区域：`Tab` 缩进，`Shift+Tab` 取消缩进。

## 消息气泡与 Tips

- 保存、删除确认、导入导出、复制反馈、区域 Tips 都通过统一的消息气泡展示。
- 气泡出现和消失有过渡动画。
- 鼠标悬浮在消息气泡上时，会暂停自动消失计时；鼠标移开后继续计时。
- 图床、快捷链接和提醒事项空白区域的 Tips 在显示期间不会因重复点击而快速切换。
- 切换到其他区域会取消当前 Tips，新区域可以显示自己的 Tips。
- 动态头像最多显示 10 秒，超过后会渐隐。

## 保存逻辑

- `Ctrl+S` 会立即保存并显示保存气泡。
- 文本输入停止 3 秒后自动保存。
- 文本区域失焦后会自动保存。
- 导入、删除、清理已完成等高风险操作都会先确认。
- 删除后的撤销入口已取消，删除确认后即生效。

## 数据位置

看板状态保存在浏览器 `localStorage`：

```text
mini-desk-state-v1
```

图片原文保存在同站点 IndexedDB：

```text
mini-desk-images-v1
```

自定义 GIF 资源也保存在同站点 IndexedDB。历史版本使用的 `todo-board-state-v1` 和 `todo-board-images-v1` 会在读取时兼容。清空浏览器站点数据会清除看板内容、图片和自定义 GIF。只删除 `localStorage` 键会清除看板状态，但 IndexedDB 中可能仍保留图片原文。

## 移动端说明

当前看板主要为桌面端工作流设计。移动端会显示引导页，提示用户在电脑浏览器打开，以获得完整体验。

## 部署

线上站点部署在 Cloudflare Pages：

- <https://minidesk.online>

Cloudflare Pages 使用 `dist` 目录部署，项目名为 `todolist`。

```bash
npm run build
```

部署到 Cloudflare Pages：

```bash
npm run deploy:cloudflare
```

发布版本时，先构建 `dist`，再将编译产物压缩为 release asset：

```bash
npm run build
cd dist
zip -qr ../dist-<version>.zip .
```

Vite 的 `base` 由环境变量控制：

```ts
base: process.env.VITE_BASE ?? "/"
```

## 项目结构

- `src/App.vue`：看板主入口，负责跨组件状态编排、保存、导入导出、消息气泡和快捷键。
- `src/components/`：图床、图片预览、工具、快捷动作、提醒事项、文本区、工作空间、设置菜单和消息气泡组件。
- `src/state/`：默认状态、消息文案、版本信息、GIF 主题、localStorage/IndexedDB 存取、导入导出归一化和待办领域逻辑。
- `src/utils/`：文本编辑缩进与行处理工具。
- `src/__tests__/`：组件渲染、状态兼容、交互契约、部署配置和消息文案测试。
